# HTML & CSS Landing Page Clone Projects

This repository contains a collection of beginner-friendly landing page clone projects created using simple **HTML** and **CSS**. The main focus of the project is to practice webpage layout, visual styling, navigation sections, forms, banners, product cards, and responsive page structure using only core web technologies.

## Project Overview

The project includes four major landing page clones:

| Project | File Path | Description |
| --- | --- | --- |
| Swiggy Clone | `swiggy-clone/swiggy.html` | Food delivery landing page with header navigation, search fields, hero section, and promotional service cards. |
| Flipkart Clone | `flipkart-clone/flipkart.html` | E-commerce homepage layout with search bar, category navigation, banners, and product spotlight cards. |
| MakeMyTrip Clone | `makemytrip-clone/makemytrip.html` | Travel booking landing page with navigation, flight search panel, fare options, and booking controls. |
| Groww Clone | `grow-clone/grow.html` | Investment platform landing page with navigation, market ticker, hero heading, CTA button, and cityscape visual. |

## Technologies Used

- HTML5
- CSS3
- Inline CSS styling
- Basic form controls
- Images, icons, and visual assets
- Static webpage structure

## How to Run the Project

No installation is required because these are static HTML pages.

1. Download or clone this repository.
2. Open the project folder.
3. Open any HTML file directly in your browser.

Example files to open:

```text
swiggy-clone/swiggy.html
flipkart-clone/flipkart.html
makemytrip-clone/makemytrip.html
grow-clone/grow.html
```
